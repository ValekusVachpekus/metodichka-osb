# Методическое пособие ОСБ

Статический сайт-методичка для **Отдела Собственной Безопасности**, размещённый на GitHub Pages.

## Структура

```
index.html              — Главная страница
style.css               — Стили
pages/
  polnomochiya.html     — Полномочия ОСБ
  proverki.html         — Проверки
  raporty.html          — Написание рапортов
  opros.html            — Опрос
  nadzor.html           — Надзорная деятельность
  vzyskaniya.html       — Меры дисциплинарного взыскания
  pervaya-pomosh.html   — Первая помощь
  normativy.html        — Нормативно-правовые документы
```

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы
3. Перейдите в **Settings → Pages** → выберите ветку `main`, папку `/ (root)`
4. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`
